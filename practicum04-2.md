Задание 2: Написать приложение для управления заметками:

1. Настройте новый проект React с помощью Create React App.
2. Создайте необходимые папки и файлы для вашего проекта: App.js, NoteList.jsx,
   NoteForm.jsx, и Note.jsx. Также создайте папку styles для CSS модулей.
3. Определите CSS модули для каждого компонента (App.module.css,
   NoteList.module.css, NoteForm.module.css, и Note.module.css).
4. Импортируйте стили и компонент NoteList.
5. Создайте базовую разметку, включающую в себя контейнер для NoteList.
6. Импортируйте необходимые зависимости и компоненты (NoteForm и Note), а также CSS модуль.
7. Используйте useState для управления списком заметок.
8. Создайте функцию listUpdate, которая добавляет новую заметку в список.
9. Отрендерите NoteForm и передайте ему функцию listUpdate через props.
10. Отобразите все заметки, используя .map() для массива noteList
    и компонент Note для каждой заметки.
11. Импортируйте зависимости и стили.
12. Установите локальное состояние для текста новой заметки.
13. Создайте функцию handleChange, которая обновляет состояние при изменении текста в поле ввода.
14. Создайте функцию handleSubmit, которая предотвращает стандартное поведение формы,
    использует колбэк для обновления списка заметок и очищает поле ввода.
15. Отрендерите форму с текстовым полем и кнопкой для отправки.
16. Импортируйте зависимости и стили.
17. Инициализируйте состояние для статуса заметки (готово/не готово).
18. Создайте функцию handleChange, которая переключает статус заметки.
19. Отрендерите заметку с условным форматированием для отображения статуса
    и чекбокс для изменения статуса.
20. Стилизуйте созданные компоненты, чтобы придать приятный вид своему
    приложению
21. Проверьте, что приложение корректно добавляет заметки.
22. Проверьте функциональность переключения статуса заметки.
23. Убедитесь, что стили корректно применяются.
